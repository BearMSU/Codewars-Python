# Codewars
This is a repository of different Codewars katas I've completed. Please enjoy.

The folders that have 8kyu, 7kyu, and 6kyu katas were my work that I had done prior to installing the CodeHub extension in my browser. I copied the work saved the work to my personal folders and then uploaded them into GitHub.
The later work is after I had installed the extension. Now Codewars automatically uploads my solutions once I submit them, which is why they are formatted differently. There is no difference in how the work was done, or
any other factor, however. It's just when the extension created new folders for when it pushes automatically from Codewars, and I decided to keep the ones I had uploaded manually separate. Enjoy!
